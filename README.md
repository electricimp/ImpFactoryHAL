# impFactory Hardware Abstraction Layer

Use the impFactory_005 table to access the hardware pins and uart objects for imp005 based impFactory hardware.

## impFactory_005

* LED_RED
* LED_GREEN
* BLINKUP_PIN
* GREEN_BTN
* FOOTSWITCH
* LCD_DISPLAY_UART
* USB_EN
* USB_LOAD_FLAG
* RS232_UART
* FTDI_UART
